## üîê What is pfSense?

pfSense is an open-source, FreeBSD-based **firewall and network security platform**.

It can act as a:

- Firewall  
- Router  
- VPN gateway  
- Network security appliance  

It is widely used in:

- Hospitals and healthcare networks  
- Enterprises and data centers  
- On-prem edge devices  
- Home labs  
- Virtualized environments (Hyper-V, VMware, Proxmox)

pfSense can run on:

- Physical hardware
- Virtual machines
- Netgate appliances
